# 🌊 波塞冬 (Poseidon)

由 [tastybento](https://github.com/tastybento) 创建和维护。

**Poseidon** 是一款为 Minecraft 打造的沉浸式水下生存体验。在一场神秘的海难后，你发现自己被困，醒来后却获得了在水下呼吸的神秘能力——但这是有代价的。就像鲨鱼一样，你必须不断移动才能生存。

欢迎来到这个世界，海洋既是你的避难所，也是你最大的威胁。从深海遗迹到发光的珊瑚城，Poseidon 挑战你在这个以海洋为唯一常态的领域中生存、建造和探索——直到你敢于踏上那片可怕、炙热的陆地。

## 🐚 游戏特色

  - 🧭 **重塑生存**：不断移动以保持活力。静止不动会招致危险。
  - 🏰 **水下王国建造**：在深海中建造覆盖着珊瑚的城堡和水生城市。
  - ⚔️ **充满挑战的遭遇**：面对无情的溺尸、海怪和深埋海底的秘密。
  - 🌋 **注水的下界**：探索一个被水淹没的下界版本。
  - 📖 **故事驱动的挑战**：完成多层次的任务，解锁神秘的书籍，拓展 Poseidon 的世界。
  - 🏝️ **自定义初始岛屿**：选择你的起点——简单的沉船、小型纪念碑，或是一个宏伟的珊瑚环绕的领地。

## 🔧 安装说明

> 推荐：与 [BentoBox 挑战插件 (Challenges Addon)](https://github.com/BentoBoxWorld/Challenges) 一起安装，以获得 Poseidon 的全部功能。

1.  **下载并安装 Poseidon 插件。**
2.  **将其放入你的 `/plugins/BentoBox/addons/` 文件夹中。**
3.  **启动你的服务器** —— Poseidon 的新世界将自动生成。
4.  **让世界预生成运行。** 此功能仅在没有玩家在线时有效。预生成的区块越多，性能越好。
5.  **登录并配置挑战**：
      * 使用 `/padmin challenges` 命令
      * 点击 🕸️ 网页图标以下载 **Poseidon 挑战包**
6.  **选择一个起始岛屿**：
      * 带珊瑚的沉船
      * 带小型纪念碑的沉船
      * 同时带有小型和大型纪念碑的沉船
      * 使用 `/padmin bp` 自定义岛屿
7.  **（可选）** 调整 `config.yml` 文件或使用管理员设置 GUI。
      * 重大更改可能需要删除数据库并重置世界。

## ⚠️ 注意事项

  - 如果岛屿未预先生成，可能会出现延迟。在邀请玩家之前，请让预生成完成。

## 玩家命令

默认的玩家命令是 `/poseidon`，或简写为 `/po`。

## Config.yml (配置文件)

config.yml 文件与其他游戏模式类似，其中有一个专门用于 Poseidon 设置的部分，然后是更通用的世界和岛屿设置。在 Poseidon 中，岛屿被称为“领域 (realms)”。Poseidon 特有的设置如下：

```
poseidon:
  air-effect:
    # 玩家离开水面而不会受到伤害的宽限时间（秒）。
    grace-period: 3
    # 处于空气中时每秒受到的伤害。
    damage: 2
    # 饮水后可免受空气伤害的时间（秒）。饮水效果等同于饮用一瓶水下呼吸药水。
    water-effect-time: 30
  # 水生生物无视波塞冬后裔（玩家）的几率（百分比）。
  # 这会使游戏更简单..
  water-mob-ignore: 50
```
