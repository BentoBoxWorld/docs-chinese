# AOneBlock 占位符

以下是可用的占位符：

| 占位符 | 描述 |
| --- | --- |
| %aoneblock_my_island_blocks_for_phase% | 你岛屿当前阶段的方块数 |
| %aoneblock_my_island_blocks_to_next_phase% | 直到下一个阶段的方块数，如果没有下一个阶段则为"无限" |
| %aoneblock_my_island_count% | 你岛屿的方块计数 |
| %aoneblock_my_island_done_scale% | 你岛屿的阶段完成比例 |
| %aoneblock_my_island_lifetime_count% | 你岛屿的终身方块计数 |
| %aoneblock_my_island_next_phase% | 你岛屿的下一个阶段 |
| %aoneblock_my_island_percent_done% | 阶段完成百分比 |
| %aoneblock_my_island_phase% | 你岛屿的阶段 |
| %aoneblock_visited_island_blocks_to_next_phase% | 你所站立的岛屿直到下一个阶段的方块数 |
| %aoneblock_visited_island_count% | 你所站立的岛屿的方块计数 |
| %aoneblock_visited_island_done_scale% | 你所站立的岛屿的阶段完成比例 |
| %aoneblock_visited_island_lifetime_count% | 你所站立的岛屿的终身方块计数 |
| %aoneblock_visited_island_next_phase% | 你所站立的岛屿的下一个阶段 |
| %aoneblock_visited_island_percent_done% | 你所站立的岛屿的阶段完成百分比 |
| %aoneblock_visited_island_phase% | 你所站立的岛屿的阶段 |
