# 选择游戏模式

不确定在你的服务器上运行哪个游戏模式？此页面比较所有官方 BentoBox 游戏模式来帮助你决定。

你也可以**同时运行多个** — 许多服务器提供两个或三个游戏模式，以便玩家可以选择他们首选的体验。

---

## 快速比较

| 游戏模式 | 设置 | 难度 | 主要挑战 | 多人游戏焦点 |
|---|---|---|---|---|
| **BSkyBlock** | 天空（浮动岛屿） | 中等 | 从虚空中的微小岛屿扩展 | 中等 |
| **AOneBlock** | 天空（虚空） | 中等 | 挖掘永远不会耗尽的单个魔法方块 | 低-中等 |
| **AcidIsland** | 天空（酸液海洋） | 中等-困难 | 天空方块加危险的酸液水 | 中等 |
| **CaveBlock** | 地下 | 中等 | 在坚实的石头世界中挖掘空间 | 中等 |
| **SkyGrid** | 天空（散布的方块） | 困难 | 从单个方块的网格中收集资源 | 低 |
| **Boxed** | 正常世界 | 简单-中等 | 完成进度来扩展你的受限空间 | 中等 |
| **Poseidon** | 海洋 | 中等 | 完全在水下生存 | 中等 |
| **StrangerRealms** | 主世界 + 颠倒世界 | 中等-困难 | 在管理危险镜像维度的同时声称土地 | 中等-高 |

## 我应该选择哪个？

### 我想要最受欢迎、最著名的体验
**BSkyBlock** — 经典天空方块是大多数玩家已经知道的。它有最大的社区、最多的资源，并且是任何服务器的安全选择。

### 我想要新颖但仍然容易接近的东西
**AOneBlock** — 这个概念（一个魔法方块）很容易解释并立即引人入胜。内置的阶段进度使玩家长期保持动力。

### 我想要天空方块但更困难
**AcidIsland** — 与 BSkyBlock 相同的感觉，但落入海洋是真正危险的。适合那些发现常规天空方块太容易的玩家。

### 我想要地下/采矿主题
**CaveBlock** — 不是在天空中构建，玩家从坚实的地面中挖掘。尽管共享大多数 BentoBox 的机制，体验感觉非常不同。

### 我想迎合高级/硬核玩家
**SkyGrid** — 散布块的世界是无情的。资源分散开来，运动是危险的，生存需要真实的技能。不建议作为具有新玩家的服务器上的唯一选项。

### 我想要与原版进度相关的东西
**Boxed** — 进度驱动的扩展机制将游戏直接连接到原版 Minecraft 目标。适合喜欢结构化进度的玩家。

### 我想要独特的视觉主题
**Poseidon** — 完全水生世界看起来很棒，玩法也非常不同。作为想要视觉上不同的玩家的次要选项很好。

### 我想要故事灵感的体验和复杂机制
**StrangerRealms** — 颠倒世界镜像维度增加了一层战略。更适合已经熟悉 Minecraft 并准备好应对更复杂内容的玩家。

## 按游戏模式的功能支持

大多数 BentoBox 插件适用于所有游戏模式。少数具有特定的兼容性要求。

| 功能 | BSkyBlock | AOneBlock | AcidIsland | CaveBlock | SkyGrid | Boxed | Poseidon | StrangerRealms |
|---|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
| Level 插件 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |
| Challenges | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |
| Warps | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |
| InvSwitcher | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |
| Border 插件 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ❌* |
| 地狱世界 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ❌** |
| 末地世界 | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |

\* StrangerRealms 有自己的内置边界系统 — 不要将 Border 插件与之一起使用。

\*\* StrangerRealms 用颠倒世界维度替换地狱。

## 运行多个游戏模式

每个游戏模式完全独立运行 — 独立的世界、独立的岛屿数据库、独立的配置。玩家可以同时在每个游戏模式中拥有岛屿。

如果你运行多个游戏模式，强烈建议安装 **InvSwitcher** 插件。没有它，玩家在所有游戏模式世界中共享相同的库存、经验和健康状况，这可能会导致混淆和利用。
