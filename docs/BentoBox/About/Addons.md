# 插件

BentoBox 本身什么都做不了 — 它是一个平台。**插件**才是让它发挥作用的东西。有两种类型：

- **游戏模式插件** — 这些创建玩家实际游戏的游戏世界（天空方块、洞穴方块等）
- **功能插件** — 这些在任何游戏模式之上添加可选的额外功能

你可以精确选择要安装哪些插件，因此可以构建你想要的确切服务器体验。

!!! info "插件放在哪里？"
    插件**不**放在你服务器的 `plugins/` 文件夹中。它们放在：
    ```
    plugins/BentoBox/addons/
    ```
    将它们放在那里后，重启服务器，它们将自动生成它们的配置文件。

## 游戏模式插件

这些创建你的玩家实际游戏的世界。至少安装一个。

| 插件 | 说明 |
|---|---|
| **BSkyBlock** | 经典天空方块 — 天空中的浮动岛屿 |
| **AOneBlock** | 从一个单一的魔法再生方块开始 |
| **AcidIsland** | 天空方块，其中大海是酸 |
| **CaveBlock** | 在坚实的地下世界中生存 |
| **SkyGrid** | 虚空中分散的单个方块 |
| **Boxed** | 一个随着你完成进度而增长的盒子 |
| **Poseidon** | 水下生存 |
| **StrangerRealms** | 在危险的镜像维度中生存 |

有关完整描述，请参阅[游戏模式](GameModes.md)，或[比较它们](../../gamemodes/Comparison.md)来帮助选择一个。

## 功能插件

这些添加可选功能，适用于任何游戏模式。混合搭配适合你的服务器的功能。

### 大多数服务器必需

| 插件 | 添加的功能 |
|---|---|
| **Level** | 根据放置的方块计算岛屿评分。包括排行榜，以便玩家可以竞争。 |
| **Challenges** | 玩家可以完成的任务列表以获得奖励。非常适合保持玩家参与度。 |
| **Warps** | 玩家可以在他们的岛屿上放置传送点标志，以便其他人可以从中央菜单传送到那里。 |
| **InvSwitcher** | 在同一服务器上的不同游戏模式之间保持每个玩家的库存、经验和健康状态分离。如果你运行多个游戏模式，强烈推荐。 |

### 生活质量

| 插件 | 添加的功能 |
|---|---|
| **IslandFly** | 让玩家在自己的岛屿上飞行。 |
| **Visit** | 让玩家通过菜单访问彼此的岛屿，而不需要传送点标志。 |
| **Border** | 在玩家的岛屿保护区域周围显示可见边界。 |
| **Chat** | 添加特定于岛屿的聊天频道，以便团队成员可以私下交谈。 |
| **Bank** | 给每个岛屿一个共享银行账户。玩家一起将钱存入其中。 |
| **CheckMeOut** | 让玩家提交他们的岛屿以获得管理员审查或社区投票。 |

### 高级/竞争

| 插件 | 添加的功能 |
|---|---|
| **Biomes** | 让玩家改变他们的岛屿的生物群落。 |
| **Greenhouses** | 玩家可以建造模拟特定生物群落内部的玻璃温室结构。 |
| **Limits** | 让管理员限制玩家在岛屿上可以拥有的每个方块或生物的数量。 |
| **MagicCobblestoneGenerator** | 圆石生成器可以配置为以设定的速率生成不同的方块（矿石、材料）。 |
| **Upgrades** | 玩家花费资源升级岛屿属性，如大小、生物生成等。 |
| **TopBlock** | 基于岛屿上放置的最高单个方块的排行榜。 |
| **TwerkingForTrees** | 一个有趣的机制，玩家可以通过跳舞（反复潜行）使树苗立即生长。 |

### 实用/管理

| 插件 | 添加的功能 |
|---|---|
| **VoidPortals** | 玩家可以通过坠入虚空来前往地狱或末地。 |
| **DimensionalTrees** | 在特定维度中生长的树木产生特殊掉落物。 |
| **ExtraMobs** | 添加额外的与生物相关的挑战和交互。 |
| **FarmersDance** | 玩家可以通过在他们附近跳舞来使作物生长。 |
| **ControlPanel** | 为管理员提供 GUI 控制面板来管理 BentoBox。 |
| **CauldronWitchery** | 玩家使用大锅制作特殊的药水以获得效果和物品。 |

## 查找和下载插件

官方插件由 BentoBoxWorld 团队维护，可以在以下位置找到：

**[github.com/BentoBoxWorld](https://github.com/BentoBoxWorld)**

从每个插件存储库的**Releases**选项卡下载 `.jar` 文件。开发（未测试）构建可从 [CodeMC CI 服务器](https://ci.codemc.io/job/BentoBoxWorld/) 获得。

!!! warning "版本兼容性"
    每个插件列出了它支持的 BentoBox 版本。下载前始终检查此内容。使用不兼容版本 BentoBox 的插件可能会阻止你的服务器正确启动。

---

## 检查已安装的内容

在游戏中，运行：
```
/bentobox version
```
这列出了 BentoBox 的版本和当前加载的每个插件及其版本。在报告 bug 或寻求支持时，请包括此输出。
